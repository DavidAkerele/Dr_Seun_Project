HyperText Markup Language (HTML)

HTML is the standard markup language for Web pages. With HTML you can create your own Website. Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages. Almost every page of web is having html tags in it to render its details in browser.
